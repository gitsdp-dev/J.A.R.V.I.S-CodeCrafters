# 🛠️ Installation & Setup Guide

Follow these steps to install and activate J.A.R.V.I.S on your computer.

## Step 1 — Install Python & VS Code

Download **Python 3.14 or above (if released)** and **VS Code** (recommended).

VS Code is preferred because of its:

* User-friendly interface
* Stability
* Extension ecosystem
* Virtual environment support

*(Very useful during the initial setup of J.A.R.V.I.S 😅)*

### Download Links

#### Python Package

https://www.python.org/downloads/

#### VS Code (Stable Release)

https://code.visualstudio.com/download

#### VS Code Insiders (Early Features & Updates)

https://code.visualstudio.com/insiders

---

The installation wizard will guide you through the setup process.

## Step 5 — Open the Project

Open the extracted folder in VS Code.

---

## Step 6 — Install Recommended Extensions

Install the Python extensions and dependencies recommended by VS Code.

VS Code may show:

* Extension recommendations
* Virtual environment setup prompts

Accept and complete the setup.

The process may take a few minutes depending on your internet speed.

---

## Step 7 — Install Project Dependencies

Open the VS Code terminal and run:

```bash
pip install -r requirements.txt
playwright install
```

---

## Step 8 — Activate Virtual Environment (If Required)

If VS Code created a virtual environment, run:

```powershell
(Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned) ; (& YOUR_JARVIS_FOLDER_DIRECTORY\.venv\Scripts\Activate.ps1)
```

*VS Code may execute this automatically.*

---

## Step 9 — Launch J.A.R.V.I.S

Run:

```bash
python main.py
```

---

## Step 10 — Configure Gemini API

When J.A.R.V.I.S starts for the first time, it will ask for a Gemini API key.

Generate one here:

https://ai.google.dev/gemini-api/docs/api-key

### Instructions

1. Click **Get API Key**
2. Create a new API key
3. Copy the key immediately
4. Enter it into J.A.R.V.I.S
5. Select your operating system
6. Click **Initialize**

J.A.R.V.I.S will remember your API key for future launches.

# ⚡ Faster Startup (Without Opening VS Code)

You do not need to open VS Code every time you want to launch J.A.R.V.I.S.

---

## Option A — Without a Virtual Environment

Open **Windows PowerShell** and run:

```powershell
cd YOUR_JARVIS_FOLDER_DIRECTORY
python main.py
```

---

## Option B — With a Virtual Environment

Open **Windows PowerShell** and run:

```powershell
cd YOUR_JARVIS_FOLDER_DIRECTORY

(Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned) ; (& YOUR_JARVIS_FOLDER_DIRECTORY\.venv\Scripts\Activate.ps1)

python main.py
```

---

## Important Note

`YOUR_JARVIS_FOLDER_DIRECTORY` may look like:

```text
C:\JARVIS
```

or

```text
F:\J.A.R.V.I.S
```

or any folder path where you extracted the project.
